# File System

## Overview
Linuxにおけるディレクトリの構造と構成

## Main Directories
- /home: ホームディレクトリ
- /etc: 設定ファイル
- /var: 可変データ（ログなど）
- /bin: 基本のコマンド実行ファイル（bashなど）
- /proc デバイスに関する情報
## Key Points
- Configuration files are often located in /etc
- Logs are usually stored in /var/log
