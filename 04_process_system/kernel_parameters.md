# Kernel_parameters
## Overview 
- GRUBの設定ファイルから変更可能
- 起動時の挙動や起動オプション
## Parameters
- acpi ASCPIサポートの有効/無効
- init 起動時の初期化デーモン（プログラム）を指定 
- systemd.unit　アクティブ化するsystemdターゲットを指定
- mem システムで使用可能なRAMの量を指定　仮想マシンなどで
- maxcpus　マルチコアプロセッサのシステムが認知するプロセッサの最大数を制限
- quiet　ブートメッセージの多くを非表示に
- vga　ビデオモードを選択
- root　ルートパーティッションを別のものに設定
- rootflags　ルートファイルシステムのマウントオプション
- ro　ルートファイルシステムを読み取り専用で初期マウント
- rv　ルートファイルシステムを読み書き両用で初期マウント
